## 📥 Installation

### 1. Clone Repository

```bash
git clone https://github.com/NV-DuyManh/CINEMA_GROUP6.git
```

### 2. Initialize the database (Required):

Access the downloaded folder and find the file :
```
 nhom_4.sql.
```
Open this file using **SQL Server Management Studio (SSMS)**.

Press **Execute (F5)** to create all tables and data templates.

### 3. ▶ Run Application

Press **F5** in Visual Studio to start the program.
